# TokenDance

Showcase of agent applications leveraging LLM tokens.

[![Live Demo](https://img.shields.io/badge/demo-tokendance.vercel.app-coral)](https://tokendance.vercel.app)

## Overview

An interactive single-page site presenting six AI agent archetypes and how they harness large-language-model tokens:

1. **Conversational Agent** -- dialogue-driven interaction
2. **Code Generation Agent** -- writing and refining code
3. **Research Agent** -- information gathering and synthesis
4. **Multi-Agent Orchestrator** -- coordinating multiple agents
5. **Tool-Using Agent** -- calling external APIs and tools
6. **Creative Agent** -- generating art, copy, and ideas

Features include a token flow animation, live demo terminal simulation, architecture diagram canvas, and real-time stat counters.

## Design & Tech

- **Stack:** Single HTML file, vanilla JavaScript, HTML5 Canvas
- **Aesthetic:** Warm gold and coral palette
- **Typography:** Playfair Display (headings) + DM Sans (body)
- **Interactions:** Animated token streams, simulated terminal, canvas diagrams
- **Deployment:** Vercel
