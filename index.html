// what is tokendance

Tokens are the atoms of intelligence

Every thought an LLM produces is a sequence of tokens — fragments of language that carry meaning, intent, and capability. TokenDance is a showcase of agent architectures that orchestrate these tokens into purposeful action.

From single-turn reasoning to multi-agent swarms, each pattern represents a different choreography of the same fundamental unit: the token.

// agent catalog

Six patterns of token choreography

💬

Conversational Agent

Multi-turn dialogue with persistent memory. Tokens carry context across exchanges, building a shared understanding over time.

context-window memory persona
💻

Code Generation Agent

Translates intent into executable code. Token sequences become syntax trees, type-checked programs, and tested solutions.

AST-aware iterative tool-use
🔍

Research Agent

Retrieves, synthesizes, and reasons over external knowledge. Tokens become queries, then analyses, then structured insights.

RAG web-search citation
🌐

Multi-Agent Orchestrator

A conductor that routes tasks to specialist agents. Tokens flow between models, each contributing its expertise to the ensemble.

routing delegation consensus
🔧

Tool-Using Agent

Extends cognition with external tools. The model generates structured calls — APIs, databases, calculators — then reasons over results.

function-call schema feedback-loop
🎨

Creative Agent

Generates novel artifacts — stories, designs, music. Token sampling temperature becomes an artistic dial between precision and surprise.

temperature style-transfer generative
